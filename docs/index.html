1 GitHub 认证

📖 第一次使用？点这里查看操作步骤 ▼
  1. 前往 GitHub 注册账号（已有账号跳过）
  2. 打开本仓库，点右上角 Fork 把仓库复制到你自己账号下
  3. 进入你 Fork 后的仓库 → Settings → Pages → Source 选 GitHub Actions → 保存
  4. 点击 生成 Token ↗，勾选 repo 权限，生成后复制
  5. 在下方填入你的仓库地址和 Token，点验证连接

2 选择硬件平台与设备型号

3 选择插件功能
